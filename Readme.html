Домашнее задание №2

  • Задание 1. Для реализации данного задания написана функция task1 с двумя параметрами. Второй параметр по умолчанию равен false.
    Причем в задании написано, что если второй параметр равен true, то функция возвращает результат, а если false, то функция выводит массив строк.
    В качестве теста был подставлен массив из строк, содержащих числительные.

  • Задание 2. Для реализации данного задания написана функция task2 с одним параметром.
    Остальные параметры берутся с помощью вызова функции func_get_args().
    Для операции деления производится проверка деления на 0.
    Данная функция протестирована на примерах:
    123 : 0 : 65 : 76 Результат: Деление на 0!!!
    300 : 5 : 3 : 2 Результат: 10
    123 - 0 - 65 - 76 Результат: 18
    task2('(', -8, 8, 256, 476, 34) Результат: неизвестная операция.

  • Задание 3. Для реализации данного задания написана функция task3 с двумя параметрами.
    В функции введены дополнительно 4 логических параметра:
    • Параметр nType определяет, является ли параметр n целым числом;
    • Параметр nNatural определяет, является ли параметр n положительным числом;
    • Параметр mType определяет, является ли параметр m целым числом;
    • Параметр mNatural определяет, является ли параметр m положительным числом;
    Если все четыре параметра принимают значение true, то происходит вывод матрицы - таблицы умножения, состоящей из n строк и m столбцов.
    Иначе выводится в качестве результата, в чем заключается некорректность исходных данных.

  • Задание 4. Для реализации данного задания написаны функции task4 и task5, которые возвращают требуемый формат даты.

  • Задание 5. Для реализации данного задания написаны функции task6 и task7, которые получают строковый параметр и возвращают строку,
    полученную из исходной с помощью встроенной функции str_replace.

  • Задание 6. Для реализации данного задания написаны функции task8,task9 и task10.
    Функция task8 получает на вход два строковых параметра(имя файла и строку, которую необходимо записать в файл).
    Данная функция инициирует создание текстового файла с указанным именем и записывает в файл строку, которая указана во втором параметре.
    Функция task9 получает на вход строковый параметр (имя файла), считывает содержимое файла и производит вывод содержимого.
    В функциях task8 и task9 при открытии файла на чтение или запись возможен вариант, когда мы не можем это сделать.
    В этой ситуации вызывается функция task10, которая предназначена для обработки исключений.
    Параметр функции $exceptionType может принимать два значения:
    • W - не удалось записать в файл
    • R - не удалось чтение из файла
    Функция выводит на экран соответствующую фразу. Для функции task8 в исключении используется вызов функции task10 c параметром W,
    для функции task9 в исключении используется вызов функции task10 с параметром R.
